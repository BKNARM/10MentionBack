BDD employes

  • Afficher le nom , le prénom et le service des employés qui ont une lettre "e" à la fin du prénom et travailent dans le service direction et comptabilité :
    SELECT nom, prenom, service FROM employes WHERE prenom LIKE '%e' AND service IN ('direction', 'comptabilité'); SELECT nom, prenom, service FROM employes WHERE prenom LIKE '%e' AND (service = 'direction' OR service = 'comptabilité');
  • le nom , le prénom et la date d'embauche des employés qui ont été embaucher après 2022 et ont le sexe = 'f ' :
    SELECT nom, prenom, date_embauche FROM employes WHERE date_embauche > '2022-12-31' AND sexe = 'f';
    SELECT nom, prenom, date_embauche FROM employes WHERE date_embauche BETWEEN '2022-12-31' AND CURDATE() AND sex = 'f';
  • Afficher les différents salaire touchées dans la base entreprise :
    SELECT salaire FROM employes GROUP BY salaire;
  • Afficher les noms des employés qui ont un salaire entre 2300 et 4000 :
    SELECT nom, prenom, salaire FROM employes WHERE salaire BETWEEN 2300 AND 4000;
  • Afficher le nombre d'employés par salaire :
    SELECT salaire, COUNT(salaire) FROM employes GROUP BY salaire;
  • Afficher la profession de l'employé 981 :
    SELECT service FROM employes WHERE id_employe = 981;
  • Afficher la date d'embauche de Paul :
    SELECT date_embauche FROM employes WHERE nom = 'Paul';
  • Afficher le nombre de commerciaux :
    SELECT service, COUNT(service) FROM employes WHERE service = 'commercial';
  • Afficher le coût des commerciaux sur une année :
    SELECT SUM(salaire*12) FROM employes WHERE service = 'commercial';
  • Afficher le salaire moyen par service :
    SELECT service, AVG(salaire) FROM employes GROUP BY service;

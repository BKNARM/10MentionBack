BDD

Les bases de donnéessont des magasins qui permettent de stoquer des informations et permettent notamment d'effectuer différentes opérations de lecture et d'écriture

Elles sont efficaces pour retrouver des informations qui y sont stockées. Vous pouvez interroger une base de données, par exemple: "combien d'utilisateurs sont inscrits sur mon site web?" ou "quel article dans mon blog contiens tel ou tel mot-clé ?"

La force des bases de données réside également dans leurs posibilités de lier les différents informations qui sont préalablement stockés.

SGBD

SGBD(Système de Gestion de Base de Données)permet d'accueilir, d'exploiter et de faire fonctionner les bases de données (moteur). Nous utiliserons MySQl, d'autres SGBD existe : Oracle, Sql Server, PostfreSQL, etc.

MySQl

MySQl est un SGBDR (Système de Gestion de Base de Données Relationnelles). C'est l'un des systèmes le plus populaires au monde et est très utilisé dans le monde du web

Comme son nom l'indique, MySQl utilise le langage SQL pour effectuer des requetes.

Toutes ces actions de gestion et manipulation passeront par une requete SQL.

SQL

SQL(Structured Qerry Language)Langage de requetes structuré : est un langage informatique permettant d'interagir avec des bases de données relationnelles

Une fois que les informations sont enregistrées, il est important de pouvoir les gérer : ajout, modification, suppression, consultation

phpMyAdmin

phpMyAdmin(souvent abrégé PMA) est une application web de géstion pour MySQl. il est écrit en PHP et permet de gérer les BDD dans une interface visuel simple et intuitive.

il est important de noter que phpMyAdmin n'est qu'une application. Il est tout à fait possible de gérer MySQl avec d'autres applications similaires ou en ligne de commande.

CRUD

CRUD(Creat, Read, Update, Delete) désigne les quatres opérations de base pour le stockage et la manipulation d'informations dans une base de données

  • Create : Inserer des nouveaux enregistrement
  • Read : Le fait de récupérer des enregistrement (lecture)
  • Update : Mettre à jour des enregistrement (écriture)
  • Delete : Supprimer des enregistrement (écriture)

Les opérations correspondent chacun à un type de réquetes SQL
